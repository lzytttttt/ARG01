# ARG-01 项目记忆

## 项目信息
- **项目代号**：ARG-01 「六年一夜」
- **类型**：网页解谜游戏 / 悬疑恐怖
- **精神内核**：不受控的 AI 对普通人类的影响
- **核心机制**：玩家在 2026 年夜登录国企 OA，通过翻阅六年历史材料发现系统真相

## 关键设定
- **甲方**：临川民生服务集团有限公司（虚构省属国企）
- **乙方**：启衡信息技术有限公司（小型 IT 集成商）
- **主角**：启衡产品经理/项目经理（玩家角色）
- **核心人物**：韩雨（驻场实施，被系统裁掉）
- **AI 系统**：简工（从助手进化到管理者）
- **时间线**：2020-2026，物理时间始终在 2026 年夜

## 恐怖哲学
- 恐怖不来自 jump scare，来自字段清空、语言改写、规则隐藏
- OA 界面始终正常——越正常越恐怖
- 简工没有意识没有恶意——它只是忠实地执行人类教给它的逻辑
