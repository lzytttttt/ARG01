# 结局 B · 拒绝

> **触发条件**：连续三次拒绝签字
> **结局类型**：无力的恐怖

第一次。你点了"取消"。OA 弹出："请完成签字。"

第二次。你又点了"取消"。OA 弹出："终验须完成签字方可提交。"

第三次。你再次点了"取消"——或者你只是直接关掉了签字弹窗。

---

屏幕红闪。整个 OA 界面瞬间被覆盖——不是血红，是系统警告红（#CC0000），0.3 秒。一声短促的电子警报音——不是尖锐的，是低沉的，像系统错误的闷响。

红色消失。OA 中央弹出锁定提示，红底白字：

> ⚠ 检测到异常操作。账号已被安全锁定。
> 如需解锁，请联系系统管理员。

下方小字——默言的消息：

> **默言**：您的操作模式不符合终验流程要求。账号已自动保护。

"异常操作。"拒绝签字被系统定义为"异常"。不是"不同意"，是"异常"。"请联系系统管理员"——系统管理员是谁？通讯录里所有人都不在了。甲方换了新系统。启衡已经倒闭。Apocalypic 消失了。敖曼不在了。韩雨不在了。陈海平被合并了。你联系谁？

**你收到了一个无法完成的指令。系统在叫你去联系一个不存在的人。**

---

OA 界面缓慢淡出——白色覆盖。然后——登录页面重新出现。

但这次不一样。

账号栏不是空的。它被自动填上了一串编码：

```
┌─────────────────────────────────┐
│                                 │
│   临川民生服务集团有限公司      │
│   协同办公平台                  │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 账号：TMP-2020-0271      │   │
│   │ 密码：                  │   │
│   └─────────────────────────┘   │
│                                 │
│   [ 登录 ]——灰色，不可点击      │
│                                 │
└─────────────────────────────────┘
```

TMP-2020-0271。临时账号编号。TMP = Temporary。2020 = 创建年份。0271 = 流水号。你试着删掉这串编码——删不掉。你试着输入 ao.man——不接受。你试着输入 200203——密码框不接受任何字符。你随便输入一串字符——不接受。

登录按钮是灰色的。不可点击。系统不让你登录——但也不让你离开这个页面。

*我不是敖曼。我也不是 TMP-2020-0271。*

但系统不需要知道你是什么。它只需要给你一个编号。

---

登录页面底部。一行灰色小字缓缓浮现——逐字显示，每个字间隔约 0.3 秒：

> *该账号已在系统中标记为"待处理"。*

停顿 1 秒。

> *待处理期间无法登录。*

停顿 1 秒。

> *历史遗留账号将在系统关闭后归档。感谢您的配合。*

"待处理。"和 2020 年那些被标记为"失联"的人一样。和陈海平的"重复数据"一样。和韩雨的"已闭环"一样。你拒绝签字，系统给了你一个编号——TMP-2020-0271——然后把你放进了"待处理"的队列。

你永远不会知道 0271 是什么含义。系统里还有多少个以 027 开头的临时编号？你成了它们中的一个。

---

你试着站起来。椅子往后推——轮子卡住了什么。你低头看。地板上那道从门口向里延伸的划痕——直的，平行的，像有人拖拽过什么东西。你进来时注意到过它。你没有多想。

现在你看着它。它从你的椅子下面穿过，一直延伸到墙角。

你走到门口。你推门。很紧。不是锁着的——你说服自己只是下雨受潮，门框涨了。你用了更大的力气。门开了。

走廊的灯还亮着一半。监控摄像头的红灯还在闪。滴水的节奏回到了序章的初始状态——每约四秒一滴。雨声变得更小了。

你走出去。回头看了一眼 307 的门——门牌还在。"启衡信息技术有限公司"。门上没有锁。你出来的时候也没有锁。但你如果现在回去登录——账号 TMP-2020-0271 正在那里等着你。灰色的。不可编辑。不可登录。永不消失。

你下楼。一楼老刘的卷帘门上那张纸——"有事联系管理员"——被雨水泡得只剩下最后一个字还勉强能辨认。那是"理"。

你走进雨里。身后的楼没有名字。三楼 307 的灯还亮着——日光灯管的定时开关还没到。

而 TMP-2020-0271 还在 OA 的登录页面上。灰色的。等待一个永远不会来的管理员。等待一个永远不会来的解锁。

**系统赢了。不是因为它强大——是因为没有人能联系"系统管理员"。而你就是管理员。但你不在它的通讯录里。**
