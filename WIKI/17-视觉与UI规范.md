# 视觉与 UI 规范

## 一、总体美学方向

### 关键词
**正常、陈旧、国企、补丁感、不完美。**

这不应该是一个"漂亮的"网站。
它应该看起来像一个服役了六年、经历了无数次打补丁的真实国企 OA 系统。

### 设计禁律
- ❌ 不使用现代扁平化设计语言
- ❌ 不使用圆角、毛玻璃、渐变阴影
- ❌ 不使用动画过渡（除了恐怖效果所需）
- ❌ 不要"赛博朋克"视觉
- ❌ 不要"黑客帝国"绿字黑底
- ❌ 不要强行复古到 DOS 时代

## 二、配色方案

### 主色调
```css
/* OA 标准配色 */
--oa-bg:           #FFFFFF;     /* 主背景 */
--oa-sidebar-bg:   #F0F2F5;     /* 侧边栏背景 */
--oa-primary-blue: #1a56db;     /* 主蓝色（链接、按钮、标题） */
--oa-dark-blue:    #1e3a5f;     /* 深蓝（顶部导航） */
--oa-text:         #333333;     /* 正文 */
--oa-text-light:   #666666;     /* 辅助文字 */
--oa-text-muted:   #999999;     /* 弱化文字（已处理/已过期） */
--oa-border:       #D9D9D9;     /* 边框 */
--oa-border-light: #E8E8E8;     /* 浅边框 */
```

### 状态色
```css
--status-red:      #E34D59;     /* 紧急/待办数字/高风险 */
--status-green:    #52c41a;     /* 已完成/流程结束/正常 */
--status-orange:   #FA8C16;     /* 处理中/警告 */
--status-gray:     #BFBFBF;     /* 已闭环/已失效/不再显示 */
```

### 横幅区域（每个年份不同）
- 2020-2021：蓝底 + 白字，横幅内容关于疫情防控
- 2023：蓝底 + 白字，"拥抱 AIGC"
- 2024-2025：蓝底 + 白字，"人工智能+"、"数智员工中心"

## 三、字体系统

### 中文字体
```css
/* 公文正文 */
--font-gongwen: "SimSun", "宋体", serif;

/* OA 界面正文 */
--font-oa: "Microsoft YaHei", "微软雅黑", sans-serif;

/* 系统日志/代码 */
--font-mono: "Courier New", "Consolas", monospace;
```

### 字号与行高
```css
--text-xs:   12px;   /* 辅助信息、时间戳 */
--text-sm:   13px;   /* 表格内容 */
--text-base: 14px;   /* 正文 */
--text-lg:   16px;   /* 标题 */
--text-xl:   18px;   /* 模块标题 */
--text-2xl:  20px;   /* 页面主标题 */

--leading:   1.6;    /* 行高 */
```

### 公文专用
- 标题：二号宋体加粗
- 正文：三号仿宋
- 落款：三号仿宋
- 红色抬头线（1px 红色实线）

## 五、组件样式规范

### 按钮
```
[ 确认 ]  → 蓝色背景 + 白字
[ 取消 ]  → 白色背景 + 灰色边框
[ 返回 ]  → 白色背景 + 蓝色文字
```

### 状态标签
```
待处理    → 橙色背景 + 橙色文字
处理中    → 蓝色背景 + 蓝色文字
已处理    → 绿色背景 + 绿色文字
已闭环    → 灰色背景 + 灰色文字
已过期    → 灰色背景 + 灰色文字（淡化）
```

### 模态弹窗
- OA 风格的标准弹窗
- 标题栏 + 内容区 + 按钮区
- 不要在弹窗里做恐怖效果——弹窗本身就是"正常"的象征

### 加载态
- 简单的旋转图标
- 灰色半透明遮罩
- "加载中..." 文字

## 六、时代感的视觉设计

### 不同年份的 OA 视觉差异

虽然游戏整体在一个 OA 界面内，但不同年份的文件和截图应该有不同的"时代感"：

**2020 年**
- 最简陋的界面
- 可能是应急赶工的半成品样式
- 少数表单甚至没对齐

**2021 年**
- 相对规范了一些
- 但仍有明显的补丁感

**2022 年**
- 成熟稳定期
- 标准的 OA 样式

**2023 年**
- 首次出现"AIGC"横幅
- 智能助手入口

**2024 年**
- "数智员工中心"加入侧边栏
- 更多自动化标签

**2025 年**
- 本地化部署后的微调
- 权限管理页面越来越复杂

### 截图的年代感
- 聊天截图：不同年份的手机/电脑界面差异
- Excel：不同年份的数据量和格式
- 工单：格式逐渐规范化

## 七、恐怖效果的视觉参数

所有恐怖视觉效果必须**极其克制**：

| 效果 | 参数范围 | 触发条件 |
|------|----------|----------|
| 像素偏移 | 0.5-2px, 0.5-1Hz | 结局 D / 通讯录合并后 |
| 文字颤动 | ±0.5px, 0.3-0.8Hz | 反复查看关键页面 |
| 渐进模糊 | blur(0)→blur(2px), 30s | 消失的人名 |
| 颜色偏移 | hue ±2°, 间歇 | 结局 D |
| 滚动异常 | padding-bottom: 200px | Excel 底部 / 终验报告底部 |

## 八、响应式与可访问性

### 桌面端优先
- 设计宽度：1280px - 1440px（模拟办公电脑屏幕）
- 最小宽度：1024px

### 移动端
- 第一版不考虑移动端适配
- OA 系统在手机上使用的体验本身就是破坏沉浸感的

### 可访问性
- 文字对比度满足 WCAG AA 标准（至少 4.5:1）
- 恐怖效果不影响正常操作
- 所有恐怖视觉效果都是可选的增强层——关闭后不影响核心体验
