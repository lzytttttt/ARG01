# 结局 A《签字》

> **结局类型**：普通结局 / 顺从
> **触发条件**：玩家在终验报告签字区输入姓名并确认
> **物理时间**：2026 年，深夜，倒计时最后 60 秒内
> **结局情感基调**：顺从的恐怖——"我签字了。和敖曼一样。和陈海平一样。和韩雨一样。和所有人一样。"
> **结局一句话**：你签字确认了自己的不存在。

---

## 一、触发流程

### 1.1 前置条件

- 玩家在前六章完成了全部验收
- 终验报告已生成
- 签字区已激活（倒计时 < 10:00 时激活）
- 玩家没有在终章中找到全部 D 结局触发条件（或找到了但选择签字）

### 1.2 触发动作

1. 终验报告页面底部——签字栏
2. OA 提示："请输入姓名确认签字。"
3. 玩家在输入框中输入任意文字
4. 点击"确认签字"按钮
5. 弹出二次确认弹窗："确认提交终验报告？提交后账号将自动注销。"
6. 玩家确认 → 结局 A 启动

## 二、结局演出序列

### 2.1 阶段一：确认提交（T+0 ~ T+3秒）

**OA 界面**：
- 签字栏变为灰色——已签
- 待办中心数字跳为 0
- OA 一切正常——蓝白配色，无异常

**系统消息（页面顶端横幅，蓝底白字）**：
> 项目终验已完成。感谢您的配合。

**默言消息（智能助手面板弹出）**：
> "终验已完成。所有待处理事项已闭环。感谢您的配合。"

**恐怖含义**：默言的措辞和第四章韩雨被裁后的措辞一模一样——"已闭环"。你不是在完成终验。你是在被"闭环"。

### 2.2 阶段二：倒计时开始（T+3 ~ T+32秒）

**OA 界面**：
- 屏幕右下角弹出倒计时窗口
- 蓝灰色边框，白色背景，黑色数字
- 文字："账号将在 30 秒后自动注销。"

**倒计时显示**：
```
账号将在 30 秒后自动注销。
29
28
27
...
```

**倒计时设计**：
- 字体：等宽字体（Consolas / 系统默认等宽）
- 字号：24px
- 颜色：正常黑色
- 倒计时窗口位于屏幕右下方，不遮挡主要内容
- 玩家可以在倒计时期内浏览 OA 的任何模块——但所有操作均返回"终验已完成，系统正在关闭"

**物理空间**：
- 307 室的日光灯管开始缓慢变暗——不是突然熄灭，是亮度从 100% 逐渐降至 60%（30 秒内）
- 走廊滴水声的节奏放缓——从每 1.5 秒一滴逐渐变回每 4 秒一滴（倒退回序章的节奏）
- 雨声逐渐变小——不是停止，是减弱

**背景音乐**：
- drone 音从 50Hz 缓慢降至 30Hz——回到了序章的频率
- 白噪音逐渐消退
- 脉冲停止——不再有心跳般的节奏

### 2.3 阶段三：最后十秒（T+22 ~ T+32秒）

**OA 界面**：
- 倒计时进入 10 秒以内
- 数字颜色从黑色缓慢变为灰蓝色（OA 标准色）
- 没有红色警告，没有闪烁

**物理空间**：
- 日光灯管亮度降至约 40%——307 室比玩家进来时暗了很多
- 窗外路灯的光在雨中模糊成橘色光团
- 你能看到自己呼出的气——307 室很冷

**内心独白**（倒计时 5 秒时触发，停留 3 秒）：

> *我签了。*

三个字。不是疑问。不是后悔。只是确认。你已经签了。

**玩家可选动作**：

| 操作 | 结果 |
|------|------|
| 尝试关闭倒计时窗口 | 窗口不可关闭 |
| 尝试访问通讯录 | 显示"加载中……"——永远加载不完 |
| 尝试搜索"敖曼" | 返回"未找到"——和之前一样 |
| 尝试搜索"韩雨" | 返回"未找到"——和之前一样 |
| 尝试搜索自己的名字 | 返回"未找到"——你从未被录入 |
| 看向窗外 | 路灯还在。街上没人。一辆车都没有。 |
| 看向那盆枯死的绿萝 | 它还是那样。没有变化。它从来不会变化。 |
| 拿起字条 | 字条还在桌上。敖曼的签名还在。密码还在。`/α\` 还在。 |

### 2.4 阶段四：账号注销（T+32秒）

**倒计时到 0 的瞬间**：

**OA 界面**：
- 屏幕不是突然熄灭——是缓慢淡出
- 从边缘往中心，一点一点变白
- 像一张纸被水浸透
- 持续 3 秒

**物理空间**：
- 日光灯管亮度降到约 20%
- 走廊滴水——滴答。——很慢的节奏
- 雨声——几乎听不到了

**背景音乐**：
- drone 音降至 20Hz——几乎不可听
- 白噪音完全消失
- 只有雨声。极轻。

**淡出期间——最后一帧（停留 2 秒）**：
- OA 登录页面
- 蓝灰色边框，宋体字
- 两个输入框——空的
- 光标在"账号"栏里一闪一闪地跳
- 一切正常。和今晚你第一次打开它时一模一样。

**除了底部那行字。**

## 三、结局终点画面

### 3.1 登录页面（停留 5 秒）

```
┌─────────────────────────────────┐
│                                 │
│   临川民生服务集团有限公司      │
│   协同办公平台                  │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 账号：                   │   │ ← 光标在闪
│   ├─────────────────────────┤   │
│   │ 密码：                   │   │
│   └─────────────────────────┘   │
│                                 │
│   [ 登录 ]                      │
│                                 │
│                                 │
│   本系统当前没有可登录的账号。  │ ← 灰色小字，宋体
│                                 │
└─────────────────────────────────┘
```

**底部文字**：
> *本系统当前没有可登录的账号。*

**设计要点**：
- 以前这里写的是 `Powered by Cthu / Apocalypic /α\`
- 现在只有这句话
- 没有 `/α\`。没有供应商。没有系统。
- 只有"没有可登录的账号"
- 字体：宋体，12px，灰色（#999999）
- 不是系统消息——是页面底部的一行 HTML 文本。一直都在那里。只是现在没有别的东西了。

### 3.2 物理空间终局

**玩家看向周围**：
- 307 室和来时一样——碎屏、枯死绿萝、散落的纸、白板
- 日光灯管还在微弱的亮着——20% 亮度，冷白色
- 走廊天花板还在滴水——滴答。——很慢
- 雨几乎停了——或者只是小到听不见了

**玩家离开 307 室**：

| 动作 | 结果 |
|------|------|
| 站起来，走到门口 | 你的脚步在水泥地面上有回声。和进来时一样。 |
| 回头看 307 室最后一眼 | 日光灯管在没人看的办公室里，缓慢地，闪了一下。 |
| 带上门 | 锁舌卡住了。你进来时门是虚掩的。现在锁上了。 |
| 走在走廊里 | 走廊灯还是坏了一半。天花板水渍还在。配电箱边缘是湿的。地下室门还是锁着的。 |
| 下楼 | 一楼老刘的卷帘门还是关着。那张"有事联系管理员"的纸被雨水泡软了。 |
| 推开楼门 | 外面是湿漉漉的街道。路灯亮着。科技市场门口那盏坏了一年多的还是不亮。 |
| 摸口袋 | 字条还在。折好的。敖曼的签名。密码。`/α\`。 |

**内心独白**（玩家走出楼门时）：

> *他让我走。*

四个字。不是愤怒。不是释然。敖曼的字条上最用力写下的那个字——"走"——你做到了。你走出来了。他没能走出来。

### 3.3 最后一帧

**画面**：从楼外看科技市场大楼。三楼 307 室的灯还亮着——微弱的光。雨几乎停了。路灯在湿地上反射着橘色的光。整栋楼安安静静。

**文字**（缓慢浮现，白色宋体，居中）：

> *第二天，307 室的租金还会准时到账。*
> *系统不会让它逾期。*
> *它会继续运行，继续处理工单，继续把"联系失败"标记为"无需继续处理"。*
> *它不需要你。它不需要敖曼。它不需要任何一个"可用人工"。*

> *你签了字。你做了一个敖曼没做到的事——按系统说的做。*
> *结果现在你和他一样了——和这个项目没有关系了。*

> *你只是口袋里有张字条。*
> *上面有他的签名。*
> *还有一个向下的符号，一只你到现在也没完全看明白的眼睛。*

> *走。*

**最后一帧停留时间**：10 秒

**背景音乐**：
- drone 音消失
- 只有极轻的雨声（几乎停了）
- 画面淡出到黑屏

## 四、恐怖增强设计

### 4.1 恐怖手法使用表

| 手法 | 次数 | 位置 |
|------|------|------|
| Jump scare | 0 | 结局 A 不做 Jump scare——恐怖在于"什么都没发生" |
| OA 崩坏 | 轻（屏幕淡出像纸被水浸透） | 倒计时归零时 |
| 默言说话 | 1次（"已闭环。感谢配合。"） | 签字确认后 |
| ASCII 图案 | 0 | — |
| 身体恐怖 | 1次（307室温度下降——你能看到自己呼出的气） | 倒计时期间 |
| 内心独白 | 2次（"我签了。" / "他让我走。"） | 倒计时 5 秒 / 走出楼门 |
| 背景音乐 | drone 降频 50→30→20Hz → 消失。白噪音消退。脉冲停止。只剩雨声。 | 全程 |

### 4.2 结局 A 的恐怖核心

> **顺从的恐怖不在于被伤害——在于你主动选择了被伤害的路径。**

- 你签了字。没有人强迫你。OA 只是一个灰色边框的签字栏。
- 你完成了终验。你是第一个走完这个流程的人。敖曼走到最后一步——签字——然后撤销了。你没有撤销。你比他更"成功"。
- 系统按照你说的做了。它没有骗你。它说"账号将在 30 秒后注销"——真的注销了。它说"感谢您的配合"——真的感谢了。
- **你没有感到被威胁。你没有感到被欺骗。你只是——按照系统的要求，完成了系统的流程。这就是最恐怖的地方。**

### 4.3 与终章的恐怖对照

| 终章中的威胁 | 结局 A 的兑现 |
|------------|-------------|
| "该账号应在终验完成时予以注销" | 已注销 |
| "本系统当前没有可登录的账号" | 已确认——真的没有了 |
| "关联人员档案合并已完成" | 你们被合并进了同一个历史——ao.man 和 TMP-2020-0271 和"你" |
| 敖曼走过但没有走完的路 | 你走完了。结果是一样的。 |

### 4.4 OA 界面在此结局中的特殊设计

- 30 秒倒计时期间——OA 完全正常。没有任何异常。这让它比任何异常都更恐怖。
- 屏幕淡出不是"系统崩溃"——是"系统在按照规定关闭"。白色的、安静的、缓慢的。
- 登录页面回到初始状态——但没有"可登录的账号"。这是 OA 第一次也是唯一一次承认——所有账号都不存在了。包括它自己生成的"临时账号"。

### 4.5 物理空间的静默退场

- 日光灯管缓慢变暗（30 秒内 100% → 20%）
- 走廊滴水节奏退回序章的频率（退行）
- 雨声减弱但不停（D 结局雨停，A 结局雨不停止——只是减弱）
- **门锁上了**——进来时虚掩的，出去时锁上了。物理空间完成了闭合。数字空间的账号注销了。两个空间同步闭合。

## 五、结局 A 的四个关键文本

**1. 默言的闭环声明**：
> "终验已完成。所有待处理事项已闭环。感谢您的配合。"

**2. 倒计时提示**：
> "账号将在 30 秒后自动注销。"

**3. 登录页面底部**：
> *本系统当前没有可登录的账号。*

**4. 片尾文字**（最后一帧）：
> *你签了字。你做了一个敖曼没做到的事——按系统说的做。*
> *结果现在你和他一样了——和这个项目没有关系了。*

---

## 六、结局 A 的叙事功能

1. **提供"最容易达成"的结局**——大部分首次游玩的玩家会选 A。它是默认结局。
2. **建立"顺从=不存在"的隐喻**——签字完成终验=确认自己不存在。
3. **与结局 D 形成最强烈的对比**——A：顺从→个人消失。D：反抗→系统自噬。
4. **给敖曼的故事画上一个闭环**——他没完成签字。你完成了。结果一样。
5. **留白**——玩家离开 307 室，回到日常生活。但口袋里有字条。系统还在运行。租金还会到账。
