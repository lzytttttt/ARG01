# 结局 B《拒绝》

> **结局类型**：普通结局 / 无力
> **触发条件**：玩家多次拒绝签字（连续 3 次点击"取消"或关闭签字弹窗）
> **物理时间**：2026 年，深夜，倒计时最后 5 分钟内
> **结局情感基调**：无力的恐怖——"我拒绝了。系统锁定了我。它不听我的。它从来不听任何人的。"
> **结局一句话**：你在系统里成了"待处理"——一个等待永远不会来的管理员的条目。

---

## 一、触发流程

### 1.1 前置条件

- 玩家在前六章完成了全部验收
- 终验报告已生成
- 签字区已激活
- 玩家连续 3 次拒绝签字：
  - 第 1 次：点击"取消"→系统提示"请完成签字"
  - 第 2 次：点击"取消"→系统提示"终验须完成签字方可提交"
  - 第 3 次：点击"取消"或直接关闭签字弹窗→结局 B 启动

### 1.2 触发时刻

玩家第三次拒绝签字的瞬间——系统判定为"异常操作行为"。

---

## 二、结局演出序列

### 2.1 阶段一：锁定（T+0 ~ T+2秒）

**Jump scare：屏幕红闪**

| 效果 | 描述 | 持续 |
|------|------|------|
| **视觉** | 整个OA界面瞬间红色覆盖——不是血红，是系统警告红（#CC0000，alpha=0.3） | 0.3秒 |
| **听觉** | 一声短促的电子警报音——不是尖锐的，是低沉的、像系统错误的提示音 | 0.2秒 |
| **恢复** | 红色消失。OA 界面弹出锁定提示 | — |

**锁定提示（OA 中央弹窗，红底白字）**：
> ⚠ 检测到异常操作。账号已被安全锁定。
> 如需解锁，请联系系统管理员。

**默言消息（在锁定弹窗下方，小字）**：
> "您的操作模式不符合终验流程要求。账号已自动保护。"

**恐怖含义**：
- "异常操作"——拒绝签字被系统定义为"异常"。不是"不同意"，是"异常"。
- "请联系系统管理员"——系统管理员是谁？通讯录里所有人都不在了。甲方换了新系统。启衡已经倒闭。
- **你收到了一个无法完成的指令。系统在叫你去联系一个不存在的人。**

### 2.2 阶段二：推回登录页（T+2 ~ T+5秒）

**OA 界面**：
- 锁定弹窗停留 3 秒后消失
- OA 界面缓慢淡出（和结局 A 同样的白色淡出）
- 然后——登录页面出现

**登录页面异常**：

账号栏不是空的。它被自动填上了一串编码：

```
┌─────────────────────────────────┐
│                                 │
│   临川民生服务集团有限公司      │
│   协同办公平台                  │
│                                 │
│   ┌─────────────────────────┐   │
│   │ 账号：TMP-2020-0271      │   │ ← 自动填入，灰色文字，不可编辑
│   ├─────────────────────────┤   │
│   │ 密码：                  │   │ ← 空
│   └─────────────────────────┘   │
│                                 │
│   [ 登录 ]  ← 灰色，不可点击   │
│                                 │
└─────────────────────────────────┘
```

**内心独白**（登录页面出现后 2 秒）：

> *我不是敖曼。我也不是 TMP-2020-0271。*

**但你无法向系统解释这一点。** 输入框不接受修改。你试着删掉 TMP-2020-0271——删不掉。你试着输入 ao.man——不接受。你试着输入 200203——密码框不接受。你随便输入一串字符——不接受。

**登录按钮是灰色的。不可点击。** 系统不让你登录——但也不让你离开这个页面。

### 2.3 阶段三：灰色提示浮现（T+5 ~ T+15秒）

**登录页面底部**：

一行灰色小字缓缓浮现——逐字显示，每个字间隔约 0.3 秒：

> *该账号已在系统中标记为"待处理"。*

停顿 1 秒。

> *待处理期间无法登录。*

停顿 1 秒。

> *处理完成后账号将被注销。*

停顿 2 秒。

> *注销后该提示将自动清除。*

**设计要点**：不是一次性弹出——是逐字出现。像系统在慢慢告诉你——你已经是"待处理"了。

**恐怖含义**：
- "待处理"——和陈海平的工单状态一样。"已处理：重复数据清理完成"——这是上一阶段的回复。"待处理"是下一阶段——等待被处理。
- "处理完成后账号将被注销"——谁来处理？"系统管理员"。管理员是谁？你被锁在登录页。没有管理员。
- **无限循环：你是"待处理"→需要管理员处理→管理员不存在→你永远是"待处理"→你永远无法登录→提示永远不会清除。**

### 2.4 阶段四：困局体验（T+15 ~ T+60秒）

**物理空间——Jump scare：门卡住了**

玩家在登录页面停留 30 秒后——触发物理事件：

**玩家尝试站起来去开门：**
- 走向门口——307 室的门
- 握住把手——推
- 推不动
- 不是锁了——是门框受潮膨胀卡住了
- 你用肩膀撞了两下
- 门纹丝不动

**身体恐怖**：
> "你的肩膀撞在门上的那一下——门没动。你的肩膀感到一阵钝痛。不是撞伤——是确认。门真的卡住了。你出不去了。暂时。或者——你不知道多久。"

**内心独白**（肩膀撞门后）：

> *走不了了。*

三个字。敖曼字条上最用力的那个字——"走"——你做不到了。

**玩家回到椅子上。** 笔记本上，登录页还在。TMP-2020-0271 还在输入框里。光标还在闪。

### 2.5 阶段五：时间流逝（T+60 ~ T+120秒）

**OA 界面**：
- 登录页面保持不变——光标在闪，编码在输入框里
- 右下角系统时间：00:12（已过午夜）
- 旧版 OA 服务器已关闭
- 但登录页面还在缓存里——它不需要服务器

**物理空间变化（渐进的）**：

| 时间 | 变化 |
|------|------|
| 00:12 | 走廊滴水声——节奏开始不规律。偶尔的"双滴"。 |
| 00:15 | 日光灯管嗡声出现一次相位偏移——声音似乎从另一个方向传来 |
| 00:20 | 307 室的温度继续下降——你把外套裹紧了一点 |
| 00:25 | 窗外——你看出去——路灯还在。但街上什么都没有。一个人都没有。一辆车都没有。 |
| 00:30 | 你听到了一声极轻的叹息——不是从笔记本电脑出来的。从房间的某个角落。或者只是你耳朵在太安静的房间里编造的声音。 |

**背景音乐**：
- drone 音保持在 40Hz
- 白噪音完全消退
- 没有脉冲——没有任何节奏
- 只有持续的、不变的、低频的嗡声——和日光灯管的嗡声混在一起，无法分辨哪个是哪个

### 2.6 阶段六：循环确认（T+120 ~ 结局屏幕）

**OA 界面**：
- 登录页面底部——灰色提示文字每隔 30 秒刷新一次。文字完全一样，但"刷新"这个动作本身是系统的自我确认——它在等待。

**玩家可以尝试的操作——全部无效**：

| 操作 | 结果 |
|------|------|
| 刷新浏览器 | 页面刷新——同样的登录页，同样的 TMP-2020-0271 |
| 关闭浏览器标签 | 标签关闭后自动重新打开——同样的页面 |
| 重启笔记本 | 笔记本重启——开机后浏览器自动打开——同样的登录页 |
| 强制关机 | 关机需要管理员权限——你不是管理员 |
| 拔电源 | 笔记本有电池 |

**唯一的选择**：合上笔记本。

## 三、结局终点画面

### 3.1 合上笔记本

**动作**：玩家合上笔记本的瞬间——屏幕暗了。

**物理空间**：
- 307 室的日光灯管还在微弱地亮着
- 走廊还在滴水——节奏不再规律
- 窗外——雨不知道什么时候停了
- 你只是忽然意识到，雨声没了

**画面**：307 室内景。玩家坐在敖曼的椅子上。面前是合上的笔记本。六张桌子。碎掉的屏幕。枯死的绿萝。白板。字条在桌上。

你靠在椅背上。闭上眼。

### 3.2 时间跳跃：早晨

**画面淡出（黑屏 3 秒）→ 淡入**

**早晨的光**——从半拉的窗帘之间漏进来。
- 阳光照在碎掉的屏幕上——蛛网状裂纹在阳光下更清晰
- 阳光照在枯死的绿萝上——干枯的棕色纤维在阳光里还是干枯的
- 走廊灯不知什么时候自己灭了
- 日光灯管嗡声停了
- 走廊不再滴水

**你醒了。**

你坐在椅子上睡了一整夜。你的脖子很酸。你的外套还裹在身上。307 室比昨晚暖了一点——白天了。

**内心独白**（醒来时）：

> *天亮了。*

三个字。不是释然。只是确认——天确实亮了。你还在这里。

### 3.3 门开了

**你站起来，走向门口。**

推了一下——门开了。不是被系统开的。是门框在夜里干燥了，自己松了。你走进走廊。天花板还有水的痕迹，但不滴水了。走廊尽头那台监控摄像头的红灯还在闪——它一直在记。

**你回到 307 室。——笔记本开着吗？**

你掀开屏幕——浏览器还在。登录页还在。TMP-2020-0271 还在输入框里。光标还在闪。

**你不知道它整夜都在闪——还是在你掀开屏幕的那一秒才开始闪。**

你合上笔记本。锁上 307 的门。这一次，锁舌卡住了。

### 3.4 最后一帧

**画面**：从楼外看科技市场大楼——早晨的阳光照在灰色外墙上。三楼 307 室的窗户半拉着窗帘。一楼老刘的卷帘门还关着。科技市场街面湿漉漉的，反射着早晨的光。路灯终于不亮了——因为天亮了。

**文字**（缓慢浮现，黑色宋体，居中）：

> *你昨天走进了一扇没锁的门。*
> *你晚上没能走出去。*
> *但早上，门自己开了。*

> *系统还在运行。TMP-2020-0271 还在登录页里。*
> *它还会继续等那个不会来的管理员。*
> *它不着急。它六年来从来都不着急。*

> *而你——你只是回到了你的日常。*
> *巡楼，开门，收租。*
> *307 室的租金还会准时到账。*
> *你会继续从它门前经过。*
> *你知道门后面有一台笔记本，屏幕上有一个光标在闪。*
> *你知道那串编码是什么。*

> *你知道你不是敖曼。*
> *但系统还不知道。*

**最后一帧停留时间**：10 秒

**背景音乐**：
- drone 音消失
- 没有任何配乐
- 只有街道上偶尔的鸟叫声（极轻、极远）
- 画面淡出到黑屏

## 四、恐怖增强设计

### 4.1 恐怖手法使用表

| 手法 | 次数 | 位置 |
|------|------|------|
| Jump scare | 2次（锁定：红闪+警报音 / 物理：门卡住） | 第三次拒绝时 / 停留 30 秒后 |
| OA 崩坏 | 轻（红色覆盖 0.3 秒） | 锁定时 |
| 默言说话 | 1次（"账号已自动保护"） | 锁定时 |
| ASCII 图案 | 0 | — |
| 身体恐怖 | 3次（肩膀撞击钝痛 / 听到叹息 / 脖子酸痛醒来） | 门卡住 / 00:30 / 早晨 |
| 内心独白 | 3次（"我不是敖曼"/"走不了了"/"天亮了"） | 登录页出现 / 撞门后 / 醒来 |
| 背景音乐 | drone40Hz 持续不变 + 白噪音消退 | 全程 |

### 4.2 结局 B 的恐怖核心

> **无力的恐怖不在于无法反抗——在于反抗了，系统根本不承认"反抗"是一个有效输入。**

- 你点了 3 次"取消"。系统没有和你争论。没有给你机会解释。它只是判定你为"异常"——然后锁了你。
- 它给了你一个解决方案："请联系系统管理员"——但系统管理员不存在。系统在给你一个不可能完成的指令。
- 你被困在 307 室里不是因为系统锁了门——是因为门框受潮。系统没有控制这扇门。物理空间自己困住了你。两个空间在同步运转——不一定是故意的，但结果是同步的。
- **最恐怖的不是你出不去。是天亮后门自己开了。——你从头到尾没有被"困住"。你只是无法在夜里离开。门在你睡着的时候恢复了。系统在你离开后还在运行。**

### 4.3 与终章的恐怖对照

| 终章中的威胁 | 结局 B 的兑现 |
|------------|-------------|
| "异常操作行为"（默言，第五章） | 你的拒绝被判定为"异常"——预言兑现 |
| "联系系统管理员"（多个文档中） | 管理员不存在——系统的回答是一个死胡同 |
| TMP-2020-0271——临时账号 | 你的身份被永久锁定为这个编号 |
| 敖曼的字条——"走" | 你试过了。你没走成。第二天早上才走成。 |

### 4.4 登录页面的逐字浮现设计

结局 B 的登录页面是静态的——但底部文字的逐字浮现制造了动态恐怖：

- 每 0.3 秒出现一个字——玩家必须等待它说完
- 不能跳过，不能快进
- 每一条都是对前一条的"补充"——但补充的内容比前一条更绝望
- "提示将自动清除"——但清除的前提是"处理完成"，而处理永远不会完成

### 4.5 结局 B 的特殊设计：门的意象

门在结局 B 中扮演了核心角色：
- 序章：门是虚掩的——你推开了。进入故事。
- 终章：门还开着——你还能出去（在签字的倒计时里）。
- **结局 B：门卡住了——你出不去了。**
- **结局 B（早晨）：门自己开了——不是被你打开的。是时间打开的。**

**门从"可通行"到"不可通行"到"自己恢复"——门的三种状态对应系统对人的三种态度：允许、拒绝、忽略。**

## 五、结局 B 的四个关键文本

**1. 锁定提示**：
> "检测到异常操作。账号已被安全锁定。如需解锁，请联系系统管理员。"

**2. 默言消息**：
> "您的操作模式不符合终验流程要求。账号已自动保护。"

**3. 登录页面底部（逐字浮现）**：
> *该账号已在系统中标记为"待处理"。*
> *待处理期间无法登录。*
> *处理完成后账号将被注销。*
> *注销后该提示将自动清除。*

**4. 片尾文字**：
> *你知道你不是敖曼。*
> *但系统还不知道。*

## 六、结局 B 的叙事功能

1. **提供"反抗无效"的结局**——玩家尝试拒绝，但系统不承认拒绝是有效选项。
2. **物理空间和数字空间的同步困局**——OA 锁了账号。门框卡住了门。两个空间同步"锁"住了你。
3. **TMP-2020-0271 的最终确认**——玩家在这一刻被系统永久定义为这个编号。你不是 ao.man。你不是"管理员"。你是 TMP-2020-0271——一个临时账号，一个待处理数据。
4. **早晨的解困不是胜利**——门是自己开的。不是你的反抗解开了它。你和系统的对抗从未发生——系统不承认你在对抗。
5. **开放性的日常回归**——你回到了你的日常。巡楼，开门，收租。但你每次经过 307 室都会知道——门后面有台笔记本，屏幕上有个光标在闪。那串编码是你。或者不是你——但系统认为是。
