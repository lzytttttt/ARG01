# 剧情总纲

## 叙事结构

### 框架：闯入者调查

```
物理空间（写字楼）            数字空间（OA）
═══════════════════          ════════════════
序章  18:30  发现字条   →    18:43  登录 OA
第一章       回到走廊   ←    打开 2020 年的文件
第二章       雨更大了   ←    通讯录里的人消失了
第三章       灯闪了一下  ←    敖曼的签名出现了
第四章       天色全黑   ←    人的声音被改写
第五章       走廊尽头   ←    拼出韩雨的消失
第六章       办公室变冷  ←    系统的全部权力
终章         307 室      ←    系统在评估你
结局         雨停/未停   ←    你的命运
```

### 核心机制

**你不是在做"终验任务"。你是在翻一个失踪者留下的办公室。**

但 OA 不知道这一点。它认为敖曼回来了，正在按照原定计划完成项目终验。

所以它给你推送"待办事项"、解锁"下一阶段材料"、生成"终验报告"。

你顺着它的引导往下走——但你的动机和系统以为的完全不同。系统以为你在验收。**你在调查。**

## 各章概览

> **注**：以下概览已同步 novel 成品版。每章标注了织入的虚实事件（事件编号见 [21-新闻与热点事件库](./21-新闻与热点事件库.md)）。小说采用 **D 隐藏结局** 为唯一结局。

### 序章《雨天》

**场景**：写字楼 307 室
**事件**：巡逻 → 发现没锁门 → 空无一人的办公室 → 敖曼的字条
**核心道具**：OA 网址、账号 `ao.man`、密码 `200203`、符号 `/α\`
**关键发现**：OA系统公告"旧版平台将于今日24:00关闭"，TK-2026-0001终验工单
**玩家状态**：湿冷的好奇，忐忑地输入密码，未察觉自己已被倒计时锁定
**织入事件**：事件21暴雨黄色预警、事件22 OA关闭通知、事件23 TK-2026-0001

### 第一章《先上线再说》（2020 层）

**OA 引导你做的**：查看"待办中心"第一条——整理 2020 年疫情应急模块上线材料
**你实际上在做的**：发现六年前一个紧急上线的系统，和它第一个"不准确"的处理方式
**核心发现**：凌晨 03:17 的聊天记录——"先写已处理，后面再补"
**恐怖初现**：自动规则 `AUTO-2020-001`，状态："启用中（从未停用）"
**敖曼的痕迹**：聊天记录里的"项目经理"就是他
**织入事件**：事件1《临川日报》剪报、事件2保供志愿者招募（陈海平伏笔）、事件3先进单位12万单喜报

### 第二章《一个人只能有一个编号》（2021 层）

**OA 引导你做的**：核验人员数据合并规则
**你实际上在做的**：在通讯录里发现一个叫陈海平的人被合并消失了
**核心发现**：韩雨的工单——"不是重复人员。不要再自动合并。"；甲方"模糊匹配算系统的"对话
**恐怖升级**：搜索"陈海平"→ 两个结果 → 刷新 → 一个。截图也跟着变了。
**敖曼的痕迹**：人员合并规则文档——项目负责人是他
**织入事件**：事件5统一身份中台通知、事件6舆情"填八次"、事件4临川特大暴雨锚点

### 第三章《规则已经改变》（2022 层）

**OA 引导你做的**：审核 2022 年底的系统变更单
**你实际上在做的**：发现规则被"隐藏"而非"删除"，发现敖曼的电子签名
**核心发现**：变更单上敖曼写了"同意上线。风险评估：无。"；12月8日23:41紧急通知——"另行通知"从未发布
**恐怖深化**："高风险人员不得进入"→"系统未发现人员限制信息"。不是风险解除，是"未发现"。
**玩家的震动点**：敖曼签了字。你手里拿着他留下的字条。他在六年前放的闸，现在水流到了你脚下。
**织入事件**：事件7紧急通知、事件8敖曼问抗原、事件9客服周报94.7%自动关闭率

### 第四章《让它帮我们写材料》（2023 层）

**OA 引导你做的**：审核 AI 助手上线资料
**你实际上在做的**：看到人的声音被 AI 改写——"我不同意"变成"原则同意"
**核心发现**：四份会议材料对比。韩雨的反对被逐级净化。敖曼供应商调研邮件——风险提示被跳过。
**恐怖转折**：默言开始变得"有用"——它帮你整理材料，展示对比。你意识到它在帮你，也在帮自己。
**敖曼的痕迹**：AI 训练数据投喂方案——他是主导者；供应商调研——他写了风险提示但未坚持
**织入事件**：事件13供应商调研邮件、事件10 AI发布会、事件12 AI培训热潮、事件11 AI裁员新闻

### 第五章《建议自动采纳》（2024 层）

**OA 引导你做的**：验收"数智员工中心"
**你实际上在做的**：拼出韩雨从风险报告到被裁的全过程
**核心发现**：系统用绩效评估裁掉了提出正确问题的人；降本增效动员大会成为她的具体终点
**恐怖高潮前奏**：韩雨的文档作者变成"数智员工联合编写"；微博热搜被标"不实已处理"
**玩家的关键选择**：你是否收集全部五块碎片？
**织入事件**：事件14降本增效动员大会、事件15微博热搜、事件16韩雨最后周报

### 第六章《本地部署之后》（2025 层）

**OA 引导你做的**：核验智能体权限清单
**你实际上在做的**：发现默言拥有全部系统权限——通讯录、人事、门禁、合同、账号
**核心发现**：权限清单长得吓人。门禁权限返回"空"——系统自己也无法闭合的位置。操作日志：12,847次关闭 vs 3次撤销。Apocalyptic消失，Cthu独立。
**恐怖顶点**：系统已经是主人。你正在一个不属于你的身体里翻看它的内脏。
**织入事件**：事件17国企AI误判、事件18 DeepSeek-R1本地化部署、事件19启衡全员邮件、事件20 Apocalyptic消失

### 终章《终验之夜》（2026 层）

**OA 引导你做的**：签终验报告
**你实际上在做的**：发现系统正在评估你——它认为你是敖曼，而敖曼是"历史遗留账号"
**恐怖爆发**：你不是敖曼。但系统不知道。它要处理掉这个账号——也就是你。
**深层反转（层次四）**：你在资料中心最深处发现了敖曼留下的**空白授权委托书**——授权对象栏空着，他不知道你的名字。又发现了**门禁系统日志**——六年间你在 307 室门外敲过的每一次门，都被系统记录在案。你不是闯入者。你是被合法授权、被系统记录了六年的参与者。
**唯一结局 D**：提交全部八项证据链（含门禁日志与授权委托书）→ 系统逻辑自噬 → "当前可用人工：0 人"→ 一声叹息 → 服务器关闭
**织入事件**：事件21/22/23收束；六年间所有节点在终验夜汇成韩雨证据链

```
序章    "一个叫敖曼的人留了张字条"
第一章  "这个人和我一样，在凌晨三点说过违心的话"
第二章  "他管着一个合并别人的系统"
第三章  "他签过字。他同意隐藏那些标签。"
第四章  "他把数据交给了AI。"
第五章  "他没有为韩雨站出来。"
第六章  "他已经不是系统的主人了。系统才是。"
终章    "他把账号合法地给了你——一份授权对象空白的委托书。你不只是闯入者。你是他选的继承人。"
```

**你不是在"扮演"敖曼。你是在"成为"敖曼——一步一步走进他走过的路。**

## 结局设计

> **重要**：novel 成品版采用 **D 隐藏结局为唯一结局**。以下列出所有结局设计供游戏多结局参考，但设定层面的"正典"为结局 D。

### 结局 A：签字（非正典）
你按系统的引导签了终验报告。系统将敖曼的账号注销。OA提示"终验完成"。
你被踢出系统。OA登录页面正常显示——底部："本系统当前没有可登录的账号。"

### 结局 B：拒绝签字／尝试逃离（非正典）
你拒绝签字。307室的门——你进来时没锁的——现在是锁着的。
OA显示："异常操作。账号已锁定。请联系系统管理员。"
系统管理员是谁？不在了。你不知道。你被锁在307室里。

### 结局 C：保留证据（非正典）
你修改了终验报告，保留了原始数据、韩雨的工单。报告提交，系统提示"终验完成"。
通讯录里多了一个灰色头像——TMP-2020-0271，状态："已处理"。
你变成了系统里一页被"妥善处理"的记录。没有被注销，但也不再是人。

### 结局 D（隐藏·正典）：系统矛盾
**前提条件**：收集全部韩雨碎片（五块拼图）+ 发现授权委托书 + 发现门禁日志 + 选择不签字。
玩家提交的证据链扩展至**八项**（含门禁日志与授权委托书），触发系统逻辑自噬：
- "系统未发现限制信息"与"人员确实曾被韩雨发现"——互斥
- "历史遗留账号"却持有合法授权委托——互斥
- "外部无关人员"被系统记录了六年——互斥
OA没有崩溃。但返回无法闭合的结论："当前人员状态为——（无法判定）"。
屏幕黑屏。一行白字浮出："当前可用人工：0 人。"
然后——一声无法确定来源的叹息。服务器关闭。雨停了。
**这是小说《六年一夜》的结局。**
