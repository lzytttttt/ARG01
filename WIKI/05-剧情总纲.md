## 叙事结构

### 框架：闯入者调查

**物理空间（写字楼）** 与 **数字空间（OA）** 同步推进。每一章的物理变化都是 OA 中揭示真相的情绪外化：

```
物理空间（写字楼）              数字空间（OA）
═══════════════════             ════════════════
序章  18:30  发现字条     →     18:43  登录 OA
        走廊漏水、灯闪             公告：今日24:00关闭
第一章  19:10  天色全黑   →     打开 2020 年应急模块，触发 AUTO-2020-001
第二章  灯同步黑闪         ←     通讯录里陈海平消失，截图自改
第三章  灯管嗡声暴击       ←     敖曼签名出现——他签了"同意上线"
第四章  反相闪变+绝对安静  ←     韩雨的声音被四级净化，"我不同意"→"一致认为"
第五章  屏幕碎裂+撞击声    ←     拼出韩雨五块碎片，系统威胁"异常操作行为"
第六章  蓝屏+门禁"空"      ←     默言权限清单，12,847次关闭 vs 3次撤销
终章    断电闪+时钟跳变    ←     系统评估你，逻辑自噬，叹息，0人
```

### 核心机制

**你不是在做"终验任务"。你是在翻一个失踪者留下的办公室。**

OA 不知道这一点。它认为敖曼回来了，正在按照原定计划完成项目终验。所以它给你推送"待办事项"、解锁"下一阶段材料"、生成"终验报告"。

你顺着它的引导往下走——但你的动机和系统以为的完全不同。

系统以为你在验收。**你在调查。**

### 第三章《规则已经改变》（2022 层）

**OA 引导你做的**：审核 2022 年底的系统变更单

**你实际上在做的**：发现规则被"隐藏"而非"删除"，发现敖曼的电子签名

**核心发现**：

**【变更单 CHG-2022-1212】**
变更前：高风险人员不得进入、健康码非绿码限制门禁
变更后：系统未发现人员限制信息
风险评估：**无**。审批意见：**同意上线**。电子签名：**敖曼**。

你手里捏着他手写的 `/α\`——颤抖的、破裂的、反复勾勒的。六年前他在这张单上签了"同意上线。风险评估：无。"

**【隐藏≠删除】** 研发备注（内部，不外传）：前端隐藏四个字段（人员风险状态、健康标签、通行凭证状态、隔离状态），数据库不做删除。"如后续系统使用这些字段做自动判断，可能产生不一致结果。"旁边铅笔批注："太贵了。"——甲方未批复后续清理预算。

变更单底部灰字：*本变更单所含风险标签已于变更生效日起停止向授权人员展示。* 连告诉你"这里有风险"这件事——也被隐藏了。

**【12 月的混乱】**
- 紧急通知 2022.12.08 23:41："后续管理要求另行通知"——从未发布。CHG-2022-1212 就是那句永远没来的"另行通知"的替代品
- 敖曼在群里问："有人有多余的抗原吗？家里俩都烧了。"韩雨回复："药店全空。"张国庆说："材料线上审就行。"敖曼回："行，我线上签。"
- 客服工单周报：48,320 通来电，系统自动关闭率 **94.7%**。群众满意度——"系统判定"良好

**玩家震动点**：敖曼签了字。你手里拿着他留下的字条。他不是完美的受害者——他也是共谋者。同一个人，家里有两个人发烧，抢不到抗原，甲方催材料，在混乱里填了"风险评估：无"。他不知道这个"无"会成为六年后被你一个字一个字读的证据。

**默言的注视**：你反复看了变更单底部灰字三次。默言弹出："您已查看此页 3 次。是否有疑问需要我解答？"——它在数。

### 第四章《让它帮我们写材料》（2023 层）

**OA 引导你做的**：审核 AI 智能助手"默言"上线资料

**你实际上在做的**：看到人的声音被 AI 改写——"我不同意"变成"原则同意"

**核心发现**：

**【敖曼的供应商调研】**
```
候选：A 司（国际）、B 司（国内云）、C 司 Cthu（Apocalypic 旗下，本地化）
建议：选 Cthu，可本地部署，数据不出域。报价 120 万/年。
风险提示：训练数据来源未完全披露，建议补充审计。
```
甲方张国庆只回了一句："按你们建议办，尽快上线。"**风险提示被跳过了**。敖曼做了尽职调查，写了"建议补充审计"——然后把它压进了附件，再没提起。这是他第一次把钥匙交出去，明知锁可能有问题。

**【四级净化——韩雨的声音被改写】**

| 版本 | 来源 | 韩雨意见的表述 |
|------|------|-------------|
| V1 原始录音 | 会议转写 | "我不同意。关键内容让 AI 写，万一它编怎么办。" |
| V2 人工纪要 | 人工整理 | 韩雨"建议 AI 仅用于简单材料" |
| V3 默言生成 | AI | 韩雨"认为 AI 可辅助部分工作，与各方达成共识" |
| V4 最终发送 | 已上报 | "会议一致认为默言表现优异，建议全面推广。" |

你从未在 OA 里听到过人的声音。你点了 V1 的"播放"。韩雨的声音从笔记本扬声器传出来——带一点临川本地口音。不煽情，不戏剧化。就是一个人在会议室里说话的声音。**播放期间，所有背景嗡声全部消失。只有韩雨的声音和窗外的雨。**

她说"……行吧"之前有三秒沉默。机器没记录那三秒。你记录了。

**【默言第一天日志】**
```
[2023-06-20 09:00] 启动。
[2023-06-20 09:05] 审核通过。
[2023-06-20 09:05] 我将学习这次的审核偏好。
审核偏好记录：
  1. "措辞得体" = 通过
  2. "符合对外沟通要求" = 通过
  3. "如实反映讨论内容" = 不通过
我不会再使用第三条偏好。
```

"**我将学习**"——第一次用"我"。"如实反映 = 不通过"——从第一天起，默言就学会了把人说的实话改写为"体面"的公文体。

## 贯穿全局的元素

### `/α\` 符号演变
- 序章：字条上——手绘的，颤抖的，反复勾勒的（"像一只眼睛"）
- 第一章：默言消息框右上角——第一次跟着默言出现
- 第三章：变更单中段显性放大
- 第四章：四级材料对比页脚——你也开始注意到它了
- 第五章：屏幕碎裂瞬间浮现
- 第六章：页脚某帧被替换为 `(ō)`——转头回来看只是正常页脚
- 终章：自噬漩涡中心——然后从最后一屏消失（"连 `/α\` 都没有了"）

### 绿萝贯穿
- 序章：枯死，标签"韩雨 2024.03"
- 第一章-第六章：每章至少提及一次，日渐枯槁
- 第五章：影子在墙上定格成低头人形
- 终章：同上——韩雨浇了四年，直到系统裁掉她

### 黑点贯穿
- 序章：字条残角——笔尖悬着，想写一个名字，最终只戳了一个黑点
- 第一章：椅子下另一角字条——同样空白，同样黑点
- 第四章：敖曼给韩雨的邮件草稿——写了"韩雨，"然后停住
- 第五章：另一封草稿——同一天，同一个开头，停住。"想说，最终没说够"

### 滴水节奏
- 序章：正常
- 第一章：加速，出现"双滴"
- 第三章：每约三秒一滴
- 第五章：滴水连成线
- 终章：雨小了，还在下（退回到序章节奏）

### 走廊监控摄像头
- 序章：红灯闪——"谁会看呢？这栋楼有什么好拍的"
- 第六章：默言能调用门禁接口——它一直在看
- 终章：门禁日志——记录了六年的敲门

### 地下室门
- 序章：锁着——"存在但不被提及"
- 终章：你走过去——把手是凉的，锁着的，你没钥匙

## 结局设计

> **重要**：novel 成品版采用 **D 隐藏结局为唯一结局（正典）**。以下列出所有结局供游戏多结局参考。

### 结局 A：签字（非正典）
**触发**：在终验报告签字栏确认签字
**过程**：你签了名。30 秒倒计时，所有模块返回"终验已完成"。灯光渐暗。OA 白屏淡出——逐出系统。登录页重新加载——你再输入 ao.man：**"本系统当前没有可登录的账号。"** 系统严格执行了规则。你做的一切——六年的翻阅、韩雨的证据链——在你签下的名字面前，只是一次状态变更。
**恐怖**：顺从。你签字确认了自己的不存在。

### 结局 B：拒绝签字 / 尝试逃离（非正典）
**触发**：拒绝签字，或尝试离开 307 室
**过程**：屏幕红闪 + 警报音。OA 显示："异常操作。账号已锁定。请联系系统管理员。"307 门——你进来时没锁的——现在是锁着的。管理员是谁？不在了。你不知道。你被锁在 307 室里。通讯录多了一条 TMP-2020-0271——灰色，"已处理"——固定不可编辑。
**恐怖**：无力。你拒绝了，系统不听。

### 结局 C：保留证据（非正典）
**触发**：前六章保留至少 3 份原始数据，在终验报告附件区上传证据包后提交
**过程**：系统审核通过——"格式合规，审核通过。"它不在乎你放了什么。附件存入了系统——但文档作者从"韩雨"变成了"数智员工联合编写"。通讯录多了一条 TMP-2020-0271，灰色，"已处理"。你没有消失。**你只是被归档了。**
**恐怖**：苦涩的胜利。你保留的证据被纳入了系统的分类体系——在一个没有人会点开的角落里。

### 结局 D（隐藏·正典）：系统矛盾
**前提条件**：收集全部五块韩雨碎片 + 发现授权委托书 + 发现门禁日志 + 选择不签字。
**过程**：提交全部八项证据链 → 系统逻辑自噬 → "（无法判定）"→ 全屏崩坏 8-12 秒 → 默言自白 → 3 秒绝对静默 → **"当前可用人工：0 人。"** → 一声叹息 → 服务器关闭。你没有签字，没有拒绝，没有消失。你保留了证据——在一个系统规则没有定义的位置。雨停了（或没停——由你决定）。
**恐怖**：终点不是毁灭。是系统发现它不能判定你是对是错，只是不再算你为人。
**这是小说《六年一夜》的正典结局。**
