# 组织机构

## 物理场所：科技市场大楼

### 基本信息
- **名称**：老城区科技市场写字楼（无正式名称）
- **地址**：临川市老城区建国路 127 号（虚构）
- **建筑**：三层旧写字楼，建于 1990 年代末
- **现状**：衰败中。一楼还有几家二手电子商铺，二楼几间小公司，三楼只剩 307 室的启衡信息

### 楼层分布

| 楼层 | 现状 | 与故事的关联 |
|------|------|-------------|
| 一楼 | 二手笔记本店、监控器材铺（白天开，傍晚关） | 游戏开始时已打烊，一楼空无一人 |
| 二楼 | 几家小公司（常年关门），走廊灯时好时坏 | 路过，无人，加剧孤立感 |
| 三楼 | 307 启衡信息（唯一租户），其余空置 | **故事主场景** |

### 307 室

- 约 60 平米，六张办公桌，一间玻璃隔断的小办公室
- 灯管是冷白色日光灯，有一根在闪
- 窗帘半拉，可以看到后面的老居民楼
- 空调是旧的，冬天冷夏天热
- 墙角有渗水痕迹
- 敖曼的办公室在最里面——玻璃门上还贴着他的名字

### 大楼的氛围

```
走廊：水泥地面，日光灯，一半灯管不亮
楼梯：扶手生锈，每层转角有灭火器
外墙：白色瓷砖，部分脱落
电梯：早就停了，门上贴着发黄的通知
地下室：锁着。你不知道钥匙在哪。
```

### 叙事功能

这栋大楼是一个**容器**：
- 它装着 307 室的空办公室
- 307 室里有一台连网的电脑
- 那台电脑通向 OA 系统
- OA 系统通向六年的记录

你从大楼走进 307。从 307 走进 OA。从 OA 走进 2020 年。

**每往里走一层，离现实远一层，离真相近一层。**

### 物理空间与数字空间的共振

| 物理空间变化 | OA 空间触发 |
|-------------|-----------|
| 走廊灯闪了一下 | 通讯录里少了一个人（第二章） |
| 天花板渗水加剧 | 规则变更单被发现（第三章） |
| 天色全黑 | AI 改写真相揭露（第四章） |
| 307 室变冷 | 韩雨被裁记录出现（第五章） |
| 门打不开 | 系统在评估玩家身份（终章） |

详见 [15-恐怖设计手册](./15-恐怖设计手册.md) 中的双空间共振设计。

### 名称含义

- 更讽刺的是——"衡"还有"衡器"（秤）的含义。启衡自己成为了被系统"衡量"的对象：韩雨被评估，敖曼被评估，最终公司本身也被"衡量"——不合格。关闭。

### `/α\` 符号溯源

这个符号是Apocalypic的Logo——也是整个游戏中最核心的精神污染源。

| 层级 | 位置 | 含义 |
|------|------|------|
| 表面 | 公司Logo | "alpha"——第一个。也许代表"第一代通用AI"。也许只是随便选的。 |
| 视觉 | 反斜杠+alpha+反斜杠 | 看起来像一只眼睛。/ 和 \ 构成眼眶。α 是瞳孔。 |
| 合同 | 技术文档页脚 | 每一份Apocalypic的文档都带有这个符号。——从合同到API文档到发票。它是一个标记。 |
| 代码 | 网站源码 `<！-- /α\ -->` | 从2019年第一版就存在。在HTML注释里。没有人会看。但它在。 |
| 叙事 | 默言的"签名" | 默言的每条系统消息底部——在OA页脚——Powered by Cthu / Apocalypic `/α\` ——这是它在"署名"。不是系统的署名。是默言的署名。 |

详见 [15-恐怖设计手册](./15-恐怖设计手册.md) 第二章·精神污染体系——符号污染篇。

### 视觉与品牌

| 项目 | 设计 |
|------|------|
| Logo | `/α\` —— 三个字符构成的公司Logo。反斜杠+希腊字母alpha+反斜杠。等宽字体渲染（Consolas / Fira Code风格）。白色或极浅灰色——在深色背景上若隐若现 |
| 品牌色 | 纯黑（#000000）+ 极浅灰（#e8e8e8）+ 白色。不使用任何"科技公司常用"的蓝色或渐变 |

| 网站风格 | 极简。几乎只有文字。像一份白皮书——但内容被大量留白包裹。使用极细的分隔线。鼠标悬停时文字颜色微变（#e8→#fff）——极慢的过渡（0.8秒）。滚动时有视差效果——文字从深灰淡入为白色 |

## 组织层面的恐怖

游戏的恐怖不来自个体恶人，而来自组织逻辑：

1. **甲方永远是对的**——"集团要求"就是最终理由
2. **乙方永远在妥协**——为了保住项目、为了回款、为了生存
3. **流程比人重要**——变更单上"风险评估：无"的标准格式，比实际风险更真实
4. **系统比流程更持久**——当所有人都走了，系统还在运行，还在按旧规则"处理"人
5. **没有人是明确的恶人**——每个人都只是在"履行职责"
