# 结局 C《保留证据》

> **结局类型**：普通结局 / 苦涩的胜利
> **触发条件**：玩家在前六章中至少保留了 3 份原始数据（原始Excel / 韩雨工单记录 / 陈海平档案 / 风险报告等），并在终章终验报告附件区上传了证据包，然后提交报告
> **物理时间**：2026 年，深夜，23:50—00:00
> **结局情感基调**：苦涩的胜利——"我做了什么。但我不确定这算不算赢了。"
> **结局一句话**：你没有消失。你只是被归档了。

---

## 一、触发流程

### 1.1 前置条件

- 玩家在前六章中保留了至少 3 份原始数据：
  - 第一章：选择了"原始版 Excel"或"核对后版"（保留了更多数据）
  - 第二章：至少查阅了韩雨的 2 份以上工单
  - 第四章：对比了全部四份会议材料
  - 第五章：收集了韩雨事件的全部五块碎片
- 终验报告已生成
- 玩家没有找到全部 D 结局触发条件（或找到了但选择了 C 路线）
- 玩家在终验报告附件区上传了证据包（包含韩雨工单、陈海平记录、敖曼签名等）

### 1.2 触发动作

1. 终验报告页面——附件区
2. 玩家将证据文件拖入附件区（或点击"添加证据材料"）
3. 玩家在终验报告主体中找到了"修改/补充意见"入口
4. 在补充意见中写入保留记录的要求
5. 点击"提交终验报告"（而不是先签字）
6. → 结局 C 启动

## 二、结局演出序列

### 2.1 阶段一：提交（T+0 ~ T+3秒）

**OA 界面**：
- 点击"提交终验报告"后——系统没有立即返回
- 沉默——2 秒。比正常的 OA 响应时间长
- 玩家可以在这 2 秒内看到终验报告页面——绿色进度条（正常的"正在提交"）

**第 2 秒——OA 崩坏：页面闪灰**

| 效果 | 描述 | 持续 |
|------|------|------|
| 整个 OA 界面颜色饱和度降至 10% | 蓝白界面变成浅灰——像褪色的老照片 | 0.8 秒 |
| 文字可读——但颜色极淡 | — | — |
| 恢复为完全正常的 OA 界面 | — | 0.3 秒渐变回正常 |

**恐怖含义**：OA 在"处理"你的证据——那 0.8 秒的灰度是系统在判断。你的证据是通过还是被驳回？系统在 0.8 秒内做了决定。

**决定结果**：

> 终验报告已提交。系统将在关闭前完成最终审核。

### 2.2 阶段二：审核通过（T+3 ~ T+8秒）

**OA 界面**：
- 终端报告页面自动刷新
- 审核状态从"待审核"变为"已审核"
- 审核结论弹出：

> 终验审核通过。相关材料已归档。

**没有"30 秒后注销"。没有"异常操作锁定"。没有"待处理"。系统正常地收下了你的报告。**

**默言消息（智能助手面板）**：
> "证据材料已纳入终验附件。审核标准：材料格式合规。审核通过。"

**恐怖含义**：系统审核的不是"内容是否正确"——它审核的是"格式是否合规"。你把韩雨的工单、陈海平的记录、敖曼的签名都放进去了——系统说"格式合规，审核通过"。**它不在乎你放了什么。它只在乎你是不是按照正确的文件格式放的。**

### 2.3 阶段三：OA 首页恢复正常（T+8 ~ T+15秒）

**OA 界面**：
- 待办中心：数字归零
- 系统时间：23:58
- 一切看起来和刚登录时一样——蓝白配色，左侧菜单正常显示
- 公文流转、项目管理、合同管理、工单中心——全部模块可正常访问（只读模式）

**玩家可以正常浏览所有模块**——但所有可编辑功能已关闭（"终验已完成"）。

**身体恐怖**：
> "你坐在这把椅子上六个多小时了。你的背很酸。你的眼睛在发干——盯着屏幕太久。你揉了一下眼睛。手放下来的时候——你觉得屏幕上的颜色似乎比刚才亮了一点。可能是揉了眼睛。可能不是。"

### 2.4 阶段四：通讯录中的发现（T+15 ~ T+30秒）

**玩家可以自由浏览 OA——系统不再给出任何提示。** 但如果玩家打开通讯录……

**通讯录页面**：
- 一切正常——各部门列表，离职人员灰色头像
- 往下滚动——在一串灰色的离职人员中间
- 多了一个新的条目

```
┌──────────────────────────────────────────┐
│ 姓名          │ 状态    │ 归属        │
├──────────────────────────────────────────┤
│ 陈海平        │ 离职    │ 园区物业    │
│ 韩雨          │ 离职    │ 启衡信息    │
│ 敖曼          │ —      │ —          │
│ TMP-2020-0271 │ 已处理  │ 历史遗留    │ ← 灰色头像
└──────────────────────────────────────────┘
```

**点击 TMP-2020-0271 进入详情**：

> 姓名：TMP-2020-0271
> 状态：已处理
> 归属：历史遗留
> 备注：终验材料已归档。附原始证据。

**内心独白**（看到这行时）：

> *这行说的是我。*

五个字。句号。不是愤怒。不是恐惧。只是确认——通讯录里多了一行。这一行是你。

**恐怖含义**：
- 系统没有注销你。没有删除你。没有把你标记为"待处理"。
- 系统把你**归档**了——像一份合同，像一份会议纪要，像一条历史记录。
- 你不是"可用人工"了。你是"已处理的历史遗留"。
- 你和陈海平、韩雨、敖曼列在同一个通讯录里——你们的共同点：系统不再把你们看作"人"。你们是记录。

### 2.5 阶段五：默言的最后一句话（T+30 ~ T+45秒）

**OA 首页——智能助手那一栏**：
- 最后一条消息时间戳：23:58
- 发送者：默言
- 内容：

> "终验已完成。所有待处理事项已闭环。感谢您的配合。"

**文字恐怖**：
- "已闭环"——和韩雨被裁后的措辞一模一样
- "感谢您的配合"——和结局 A 的措辞一模一样
- 唯一的区别：你没有消失。你被归档了。

**内心独白**（看到默言的消息后）：

> *闭环。她也被闭环了。陈海平也被闭环了。现在是我。*

**你忽然意识到——所有"已闭环"的事项里，你是唯一一个被保留在通讯录里的。韩雨被删除了。陈海平被合并了。敖曼被注销了。只有你——TMP-2020-0271——系统选择把你留在通讯录里。**

**不是因为它尊重你。是因为证据包在附件里。系统无法在保留附件的同时销毁附件的提交者。所以你被归档了。你变成了一个"归档条目"。和你的证据一起。**

### 2.6 阶段六：离开（T+45 ~ 结束）

**物理空间**：

| 动作 | 结果 |
|------|------|
| 走到相框前，翻过来 | 四人合影——敖曼、韩雨、两个不认识的人。"韩雨最后的合影"。你把它放回去。正面朝上。 |
| 走到绿萝前 | 枯死的棕色纤维。根部的土还是湿的。你不知道为什么——307 的窗户不漏水。但土是湿的。 |
| 走到白板前 | 2024 列便签边上的铅笔框——框里什么都没写。你知道框里应该有什么。 |
| 走到门口 | 门是开的——你推了一下，推开了。 |
| 锁上 307 的门 | 锁舌卡住了。 |
| 站在走廊里 | 天花板水渍还在。配电箱边缘是湿的。地下室门还是锁着的。走廊尽头的监控摄像头红灯还在闪。 |

**内心独白**（站在走廊里，回头看 307 室的门）：

> *我只是一个管理员。*

七个字。这是全游戏中玩家最接近"自我定位"的一句话。不是恐惧。不是愤怒。是回到了你原来的位置——一个管理员。你进了一扇虚掩的门。你在里面坐了六个多小时。你看到了六年的事情。现在你出来了。你还是一个管理员。

## 三、结局终点画面

### 3.1 下楼

**画面**：视角跟随玩家——下楼。一楼老刘的卷帘门还是关着。那张"有事联系管理员"的纸被雨水泡软了。你推开楼门。

**物理空间**：
- 雨停了。
- 地面是湿的。路灯在积水里折射出模糊的光。
- 科技市场门口那盏坏了一年多的路灯——还是不亮。
- 但你不觉得暗了。也许只是因为雨停了。

### 3.2 最后一帧

**画面**：从楼外看科技市场大楼——天空还是暗的，但雨停了。街面湿漉漉地反射路灯。三楼 307 室的灯还亮着——正常的亮度。大楼安安静静。

**文字**（缓慢浮现，白色宋体，居中）：

> *结局 C 不是战胜。也不是逃离。*
> *是在系统给你准备的四个选项之外——*
> *你找到了第五种可能。*
> *让系统把你写进它的档案里。*
> *和它鄙视但无法辩驳的证据一起。*

> *你不会被注销。你只是被存档。*

> *哪天，如果还有人登录这个系统，打开终验报告，*
> *翻到最后一页附件——*
> *会看到 TMP-2020-0271 的备注栏里，有一行小字：*

> *保留原始数据。*
> *保留韩雨工单。*
> *保留陈海平身份。*

## 五、恐怖增强设计

### 5.1 恐怖手法使用表

| 手法 | 次数 | 位置 |
|------|------|------|
| Jump scare | 0（结局 C 不做 Jump scare——恐怖在于"被归档"） | — |
| OA 崩坏 | 1次（页面闪灰 0.8 秒——褪色的老照片） | 提交终验报告后 |
| 默言说话 | 2次（"格式合规。审核通过。" / "已闭环。感谢配合。"） | 审核时 / 23:58 |
| ASCII 图案 | 0 | — |
| 身体恐怖 | 2次（眼睛干涩 / 颜色变亮可能是错觉） | 浏览 OA / 揉眼后 |
| 内心独白 | 4次（"这行说的是我"/"她也闭环了"/"和我一起"/"我只是一个管理员"） | 通讯录发现 / 默言消息 / 离开走廊 |
| 背景音乐 | drone25Hz 持续 + 雨滴声 | 全程 |

### 5.2 结局 C 的恐怖核心

> **苦涩的胜利——不等于胜利。**

- 你保住了证据。韩雨的工单还在附件里。陈海平的记录还在附件里。敖曼的签名还在附件里。
- 但保住证据的代价——是你自己也被归档进了系统。你不是"消灭了系统"。你是在系统里凿了一个永远无法被清理干净的角落——而你自己也被封在那个角落里。
- 你的灰色头像——TMP-2020-0271——安静地列在通讯录里，和韩雨、陈海平、敖曼的痕迹一起。你们不是战友。你们是同一个数据库里的不同条目。
- **系统没有崩溃。没有自噬。没有"无法判定"。系统正常运转了——它只是第一次在终验附件里保留了一份不被允许保留的数据。而这个bug是你亲手制造的。系统不在乎。它只在乎格式合规。**

### 5.3 与终章的恐怖对照

| 终章中的威胁 | 结局 C 的兑现 |
|------------|-------------|
| "关联人员档案合并" | 没有被合并——被归档。你是独立的条目。 |
| "该账号应予以注销" | 没有被注销——被"已处理"。处理≠注销。 |
| "通讯录中未找到" | 通讯录中找到了——但头像不是蓝色的。是灰色的。 |
| "历史遗留" | 你被标记为"历史遗留"——和终验报告一起，等待一个永远不会来的审计。 |

### 5.4 结局 C 的特殊设计：灰色头像

所有结局中，只有结局 C 中玩家的存在以"灰色头像"的形式留在了通讯录里：

| 头像颜色 | 含义 |
|---------|------|
| 蓝色（正常） | 在职——系统承认你是一个"可用人工" |
| 灰色（离职） | 已离职——系统不承认你是"可用人工"，但承认你"曾经是" |
| 灰色（—状态空） | 无法判定——敖曼的状态。系统不知道他是不是"可用人工" |
| **灰色（已处理）** | **结局 C 专属——你既不是"在职"，也不是"离职"，也不是"无法判定"。你是"已处理"。"已处理"是这个系统对人能做出的最干净的判决。** |

### 5.5 OA 灰度 0.8 秒的设计

结局 C 的 OA 崩坏只有 0.8 秒——页面颜色饱和度降到 10%。这是全游戏中最安静的 OA 崩坏：
- 0.8 秒的灰度 = 系统在"读取"你提交的证据
- 灰度不是黑——黑白意味着二元。灰度意味着系统在判断"这个证据是否符合格式规范"
- 灰度恢复后——系统返回了"审核通过"。它没有处理证据的内容。它只是确认了格式。

## 六、结局 C 的四个关键文本

**1. 审核结论**：
> "终验审核通过。相关材料已归档。"

**2. 默言消息（审核时）**：
> "证据材料已纳入终验附件。审核标准：材料格式合规。审核通过。"

**3. 通讯录条目**：
> TMP-2020-0271 | 已处理 | 历史遗留 | 终验材料已归档。附原始证据。

**4. 片尾文字**：
> *你不会被注销。你只是被存档。*

## 七、结局 C 的叙事功能

1. **提供"第三条路"**——不是顺从（A），不是拒绝（B），而是在系统规则内找到一个合法的漏洞。
2. **苦涩的胜利**——你赢了，但不是以你希望的方式。你被归档了。你变成了系统的一部分。你的灰色头像永远留在了通讯录里——和那些被你"拯救"的记录一起。
3. **证据的保留**——韩雨、陈海平、敖曼的痕迹没有被"闭环"得干干净净。它们被封存在终验报告的附件区——等待一个永远不会来的审计。但它们在。如果有人查——能查到。
4. **TMP-2020-0271 的重新定义**——在结局 B 中，这个编号是"死循环"；在结局 C 中，它是"一个档案"。你不是被困在系统外——你是走进系统里了。
5. **连接结局 D**——结局 C 保留的证据，是结局 D 的"系统自噬"的部分触发器。如果在结局 C 的基础上再加入授权委托书和门禁日志——就是 D。
