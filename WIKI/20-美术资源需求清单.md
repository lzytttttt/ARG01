## 零、仅有的 2 张 PNG —— 敖曼的手写字条

### PNG-01 — 字条正面

| 属性 | 要求 |
|------|------|
| 为什么必须是 PNG | 手写字迹的力度变化、纸张纹理、撕边的不规则、涂改痕迹——CSS 和 SVG 都无法在 40 分钟游戏时长内持续提供"这是一张真实的纸"的幻觉 |
| 规格 | 1200×1600，300dpi 等效分辨率 |
| 内容 | 参见 [00-序章-雨天](./00-序章-雨天.md) 第五节 |
| 生成方案 | 找一段真实手写样本作为基底，用代码（如 Pillow+Cairo）合成文字和纸张纹理；或纯手写后扫描 |
| 降级方案 | 如果连这 1 张 PNG 也无法生成——用 CSS 模拟：米白背景 + `border: 1px dashed` 模拟撕边 + 手写体 web font + `text-shadow` 模拟笔迹粗细变化 + `transform: rotate()` 制造不均匀排列。效果会打折扣但可接受。 |

### PNG-02 — 字条背面

| 属性 | 要求 |
|------|------|
| 规格 | 1200×1600 |
| 内容 | 合同打印片段 + 底部灰色小字 + 黑点 |
| 降级方案 | 和正面一起做，或也用 CSS 模拟（打印文字用衬线字体即可） |

## 一、场景与氛围（纯代码）

### S01 — 科技市场大楼外部

```
实现方式：CSS 绘制的建筑立面
```

用 CSS 画一栋三层旧楼：

```css
/* 思路 */
.building {
  /* 米灰色外墙，瓷砖纹理用 repeating-linear-gradient */
  background: 
    repeating-linear-gradient(0deg, #d4c9b8 0px, #d4c9b8 48px, #c4b9a8 48px, #c4b9a8 50px);
  /* 斑驳用径向渐变叠加 */
}
/* 窗户：CSS grid 排列的深色矩形 + 内发光模拟室内暗影 */
/* 招牌：CSS 绘制的褪色灯箱 */
/* 天空：linear-gradient 灰白阴天渐变 */
/* 雨丝：CSS animation 的斜向半透明线条 */
/* 积水：底部镜面反射的渐变块 */
```

- 不需要照片级真实——需要的是**绘画式的氛围感**
- 天空用 `#8a8a8a → #c0c0c0` 的渐变模拟阴天
- 雨的动画：`@keyframes rain` —— 一组斜向移动的半透明线条

### S02 — 一楼走廊

```
实现方式：CSS 3D 透视走廊
```

- 灰绿色墙面（`#a8b5a2`）
- 日光灯管用细长白色矩形 + `box-shadow: 0 0 20px rgba(255,255,240,0.3)` 模拟冷光
- 灭掉的灯管用灰色矩形
- 卷帘门用垂直条纹渐变
- 电梯门上"停运通知"——一张黄纸，用 rotated div + 宋体文字
- 地面反光用轻微的镜面渐变

### S03 — 三楼走廊 + 307 室门口

```
实现方式：CSS 走廊 + 焦点在尽头的门
```

和 S02 同一套 CSS，但：
- 灯光更暗（降低所有亮度值）
- 天花板渗水：用 `radial-gradient` 画水渍斑点，`opacity` 随时间缓慢增大
- 307 的门牌：SVG 小标签
- **关键**：这个场景在游戏中反复出现。每次回来，`opacity` 微调、水渍 `transform: scale()` 微增——玩家不一定注意到变化，但潜意识会。

### S04 — 307 室全景

```
实现方式：CSS 绘制的等距或正面视角办公室
```

这是玩家待得最久的"物理空间"。用 CSS 构建：
- **六张桌子**：CSS 矩形 + `box-shadow` 模拟桌面厚度。灰白色桌面。
- **五台显示器**：CSS 矩形。三台亮着（屏幕区域用浅蓝灰 `#c8d6e5` 模拟待机状态）。两台碎裂——用 `clip-path: polygon()` 切出不规则碎片 + 裂缝用黑色细线（`border` 或 `::after` 伪元素）
- **键盘**：CSS 绘制的细长矩形 + 按键网格（不需要真的每个按键，缩略感即可）
- **杯子 + 咖啡渍**：SVG 椭圆 + `filter: blur()` 模拟干涸液体痕迹
- **枯萎盆栽**：SVG 曲线绘制干枯的茎叶，颜色 `#6b5b3a`（干褐色）
- **散落的纸张**：3-5 个 `div`，白色矩形 + `transform: rotate()` 随机角度
- **窗帘**：CSS 半透明渐变 + 轻微 `animation` 飘动
- **玻璃隔断办公室**：半透明矩形 + `border` + 反光高光条
- **墙面渗水**：同 S03 的水渍斑块

整体光照：使用一个覆盖层的 `div`，`background: rgba(30,35,50,0.15)` 暗角效果，模拟阴天室内。

### S05 — 敖曼办公室（玻璃隔间内）

```
实现方式：S04 的子区域放大 + 聚焦
```

- 玻璃门上的名牌：SVG 小矩形，"敖曼 产品经理"用宋体
- 台灯暖光：`radial-gradient(circle, rgba(255,240,210,0.4), transparent)` 覆盖层——整个 307 唯一的暖色光源
- 电脑屏幕亮着：蓝色色块，上面有 OA 登录页的缩略表示
- 桌上的字条：白色矩形 + 些许歪斜——但你不需要在这里看清字条内容，这里只是"发现字条"的视角。点击后弹出 PNG-01 的大图。

### S06 — 窗外雨景（3 个时间变体）

```
实现方式：CSS 窗口 + 3 套渐变参数
```

窗口本身是一个 `div` + `border`。窗外的内容用 CSS 渐变：

| 时间 | 天空渐变 | 建筑轮廓 |
|------|----------|----------|
| 傍晚（序章-第二章） | `#7a8a9a → #b0b8c0` | 深灰矩形 `#4a4a50`，窗户里零星灯光 |
| 入夜（第三-五章） | `#3a3a4a → #5a5a6a` | 同建筑，灯光减少 60% |
| 深夜（第六章-终章） | `#1a1a28 → #2a2a38` | 几乎全黑，只剩一两个窗户还亮着 |

雨水动画（共用）：
```css
@keyframes rainOnWindow {
  0%   { transform: translateY(-100%) translateX(0); opacity: 0.6; }
  100% { transform: translateY(100vh) translateX(-20px); opacity: 0; }
}
```
多条斜向白线，不同 `animation-delay`，模拟雨水在窗户上流淌。再加上 `filter: blur()` 的随机水珠。

纯 HTML/CSS 的数字显示：
```html
<div class="countdown">⏱ 00:23:41</div>
```
灰色、小字、OA 界面角落。用 JS `setInterval` 驱动递减。

### H05 — 文字微颤
```css
@keyframes textJitter {
  0%, 100% { transform: translate(0, 0); }
  25%      { transform: translate(0.3px, -0.3px); }
  50%      { transform: translate(-0.2px, 0.2px); }
  75%      { transform: translate(0.1px, 0.1px); }
}
```
DOI：0.5-1Hz，持续触发。应用到关键页面文本。

### H06 — 渐进模糊
```css
.text-fading {
  animation: slowBlur 30s forwards;
}
@keyframes slowBlur {
  from { filter: blur(0); }
  to   { filter: blur(2px); }
}
```
应用于"已经消失的人"的文本。

### H07 — 颜色偏移
```css
@keyframes hueDrift {
  0%, 100% { filter: hue-rotate(0deg); }
  50%      { filter: hue-rotate(1.5deg); }
}
```
极缓慢的色相漂移——玩家不一定注意到，但潜意识会不安。

## 八、给"字条"的降级方案

如果连 2 张 PNG 也无法生成，终极降级：

```html
<!-- 用 CSS 伪造一张手写字条 -->
<div class="handwritten-note">
  <p class="note-line" style="transform: rotate(-1.2deg)">如果你看到了这个——</p>
  <p class="note-line" style="transform: rotate(0.3deg)"></p>
  <p class="note-line" style="transform: rotate(-0.7deg)">我叫敖曼。启衡信息的产品经理。</p>
  <p class="note-line" style="transform: rotate(0.8deg)">这个项目我做了六年。</p>
  <!-- ... -->
  <p class="note-symbol" style="transform: rotate(-2deg); font-size: 22px;">/α\</p>
  <p class="note-end" style="transform: rotate(-1.5deg); font-size: 20px; font-weight: bold;">走</p>
</div>
```

```css
.handwritten-note {
  background: #f5f0e8;                    /* 米白纸张 */
  background-image:                        /* 纸张纹理噪点 */
    repeating-linear-gradient(0deg, transparent, transparent 2px, 
      rgba(180,160,120,0.03) 2px, rgba(180,160,120,0.03) 3px);
  border: 3px dashed #c0b0a0;            /* 模拟撕边 */
  border-radius: 2px;
  padding: 30px 25px;
  max-width: 500px;
}
.note-line {
  font-family: 'ZCOOL KuaiLe', 'Ma Shan Zheng', cursive;  /* 手写体中文字体 */
  font-size: 16px;
  line-height: 2;
  color: #2a2a2a;
  text-shadow: 0.3px 0.3px 0 rgba(0,0,0,0.2);            /* 模拟笔触粗细 */
}
```

**效果：可以达到真实字条的 60-70%。** 恐怖感会有所损失——因为"真实感"是字条恐怖的核心——但可接受。

推荐使用的中文手写 web font（Google Fonts，免费）：
- `ZCOOL KuaiLe`（潇洒体）
- `Ma Shan Zheng`（马山正）
- `Liu Jian Mao Cao`（流江毛草）

## 九、技术实现策略

### 核心思路

```
游戏 = 一个 HTML 页面
     ├── #scene-container    （物理空间：走廊/307室/窗外）
     │   └── 全部 CSS + SVG 绘制
     └── #oa-container       （数字空间：OA 系统）
         └── 全部 HTML + CSS 表单/模板
```

两个容器互斥显示（或半透明叠加），通过 JS 切换。

### CSS 变量体系

```css
:root {
  /* OA 色彩 */
  --oa-blue: #1a56db;
  --oa-bg: #ffffff;
  --oa-text: #333;
  --oa-border: #d9d9d9;
  
  /* 场景色彩 */
  --scene-wall: #a8b5a2;
  --scene-floor: #8a8a80;
  --scene-shadow: rgba(20,25,35,0.15);
  
  /* 恐怖参数（JS 可动态修改） */
  --horror-flicker: 0;
  --horror-blur: 0px;
  --horror-shift-x: 0px;
  --horror-shift-y: 0px;
  --horror-hue: 0deg;
}
```

游戏引擎通过修改 CSS 变量来驱动所有恐怖效果。

### 不需要的库

- ❌ Three.js / Canvas 游戏引擎
- ❌ 图片精灵表
- ❌ 预渲染视频
- ❌ 任何需要 GPU 重度运算的东西

需要的只是一个现代浏览器对 CSS animation / filter / SVG 的支持——Chrome/Edge/Firefox 最新版即可。
